Executive Summary

The goal of this project is to create an annual returns data set that will be used in future projects. The data set includes geometric annual returns, CRSP permno, ticker, cusip, and market capitalization.

Creating the Annual Returns Data Set

WRDS was used to query the datasets required for these calculations. The stock universe included all stocks CRSP has within their database from 1925-2017. The CRSP DSF dataset was used to find the daily returns for each stock and this was used to calculate the geometric annual returns. The CRSP DSE data set was used to attain all tickers from the CRSP permno. The cusip and permno were both included in the CRSP DSF dataset. To get the market capitalization the avg of the bid and ask prices on the closing day of the year was multiplied by stocks outstanding.
